What is TFT Ally?

TFT Ally is an Overwolf-based desktop companion that helps you improve at Teamfight Tactics. It surfaces publicly available Riot API data — meta compositions, player stats, match history, and item recipes — in a clean, accessible interface. No hidden information. No automation. Just better data for better decisions.

Key Features

Meta Tracker

Live S/A/B/C tier lists showing which compositions are performing across the ranked ladder.

Player Scouting

Search any summoner, view 20-match history, analyze comp preferences, top-4 rate, and placement trends.

Analytics Dashboard

Visualize your performance with win-rate-by-comp charts, placement timelines, and augment heatmaps.

In-Game Overlay

Lightweight toggleable overlay (Ctrl+T) with live meta recommendations and an item cheat sheet.

Tech Stack

Built with React + TypeScript + Tailwind CSS on the Overwolf platform. Riot API calls are proxied through Supabase Edge Functions for secure key management. All match data comes directly from Riot's official TFT API.

Open Source

TFT Ally is being developed in the open. Track progress, report issues, or contribute on GitHub.
